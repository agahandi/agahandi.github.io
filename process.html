Design process

Rationale for design selection

Nielsen's 10 usability heuristics

In order to design a user-friendly and responsive portfolio website, I employed Neilsen's usability heuristics and Bootstrap classes. According to Nielsen, design should match the real world, and the language should be simple, straightforward and engaging. Therefore, I used simple, intimate, and informal vocabulary so that a wide range of users can understand it. Nielsen also points out that all design sections must be consistent to avoid making the user memorize the visual information of different parts. Therefore, the colour and fonts are the same on the home and inner pages. Another way to prevent the user's memory from being loaded with unnecessary information is to apply minimal design to the website. For example, Artwork can be a part of the resume page, but I sufficed to mention photography and quilling on this page and considered a separate section for more details. To be specific, I used bootstrap and javascript to design this section of the navigation bar as a dropdown item so that the subsection contents, photography and quilling, can be explained independently, and the user can find only one topic on each page.Finally, the design should assist the user in finding and solving the errors. Therefore, if the email and message boxes are empty on the contact page, the error message will be displayed to remind the user to fill in all the necessary sections.

Readability and Visibility

Another Standard of UX design is the legibility of the text. Therefore, I evaluated the contrast ratio between the text and background colour by using webaim.org. According to this website, the contrast ratio between colours #ffffff and #006666 is 6.79:1 for large texts, which can pass WCAG 2.0 level AAA. Thus, all heros' headings and subheadings' font weight is bold. Also, I chose the "Lato, sans-serif" font family because this font is simple and legible in most sizes and weights. Besides, CSS {line-height: 2rem;} is considered for paragraphs with smaller font size to make the words more visible. Since the wordmark and inactive navigation items are black, I chose a lighter color, viridian green, compared to bg-dark color for logo and active items.
